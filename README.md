# Savant

A Maven companion tool.

## Getting Started

```bash
$ git clone git@github.com:spilth/savant2.git
$ cd savant2
$ mvn package
$ alias savant="java -jar ./target/savant-2.0.0-SNAPSHOT.jar"
```

Currently only a basic `init` command is supported and initializes a 

```bash
$ savant init hello
$ cd hello
$ mvn package 
```
